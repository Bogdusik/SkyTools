# Screenshots Directory

Place your app screenshots here:

- `dashboard.png` - Dashboard view with telemetry
- `map.png` - Flight map with track
- `logs.png` - Flight logs and sessions
- `system.png` - System status view
- `settings.png` - Settings screen

## Recommended Screenshot Sizes

- **iPhone 14 Pro / 15 Pro:** 1179 x 2556 pixels (portrait)
- **iPad Pro:** 2048 x 2732 pixels (portrait)

## Tips

- Use Xcode Simulator or real device screenshots
- Ensure screenshots show the app in a good state (connected drone, active session)
- Keep file sizes reasonable (< 500KB per image)
- Use PNG format for best quality
